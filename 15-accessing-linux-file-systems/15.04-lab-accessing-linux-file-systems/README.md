## Цели

Проверка базовых навыков монтирования файловых систем и поиска файлов

## Задачи

В этой лабораторной работе вы смонтируете локальную файловую систему и найдете в ней определенные файлы.


## Инструкции

Войдите на **workstation** как пользователь *student* с паролем *student*.

На **workstation** выполните команду `lab fs-review start`. Эта команда запускает подготовительный сценарий, который проверяет доступность хоста **serverb** в сети. Сценарий также создает раздел на втором диске, подключенном к **serverb**.

```
[student@workstation ~]$ lab fs-review start
```

1.	На **serverb** от имени пользователя *root* определите UUID устройства **/dev/vdb1** и смонтируйте **/dev/vdb1** по его UUID в каталог **/mnt/freespace**.
2.	Создайте отчет об использовании диска в каталоге **/usr/share** и сохраните результаты в файле **/mnt/freespace/results.txt**.
3.	Используйте команду `locate`, чтобы найти все файлы конфигурации **rsyslog.conf** и сохранить результаты в файл **/mnt/freespace/search1.txt**.
4.	Сохраните результаты поиска всех файлов в каталоге **/usr/share**, размер которых больше 50 МБ и меньше 100 МБ, в файл **/mnt/freespace/search2.txt**.
5.	Выйдите с **serverb**.

## Оценка

На **workstation** запустите сценарий `lab fs-review grade`, чтобы проверить, правильно ли была выполнена лабораторная работа.

```
[student@workstation ~]$ lab fs-review grade
```

## Конец

На **workstation** запустите сценарий `lab fs-review finish`, чтобы закончить упражнение.

```
[student@workstation ~]$ lab fs-review finish
```

Лабораторная работа завершена.
